FAQ

Will we be allowed to have more widgets later?

Yes! Right now, it's limited to one because of the system we're using, but the intention is to allow up to 4. We will roll this update out as soon as we can!

My homepage keeps breaking when I remove and add a widget! Why?

Well, if we're getting real technical, it's because of the way the homepage is designed. It uses IDs to track which section an app icon is in, and it loads it's position accordingly. However, whenever we remove and replace the widget, the code destroys some icons and replaces them where they need to be, subsequently breaking the layout. This is a limitation in the system we're using that we intend to patch later on. For now, you will have to reset your homepage layout every time you create or remove your widget. We will let you know when a fix comes out!

Will there be more shapes?

The answer is a strong maybe. More shapes are not on the to-do list, but if you have any recommendations, please let us know and we will consider them! However, it's a lot harder than you might think.
